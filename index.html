⚡ Quick Start — Anaglyph Mode
  1. Patient opens Anaglyph View and wears Red-Cyan glasses.
  2. Doctor opens Controller, enters the room code, taps Connect.
  3. Select Anaglyph display mode and choose a target (Cross in Cage recommended).
  4. Confirm patient sees both corner markers (▲ red + ◆ cyan) — verifies no suppression.
  5. Select Base-Out, tap + step by step.
  6. Record Break on diplopia, Recovery when fusion returns.
  7. Repeat with Base-In.
